# Ruby - PITA
```
git clone https://github.com/rbenv/rbenv.git ~/.rbenv
git clone https://github.com/rbenv/ruby-build.git $(HOME)/plugins/ruby-build
```
# restart shell
```
~/.rbenv/bin/rbenv init
rbenv install 3.3.1
rbenv global 3.3.1
```

pyenv install 3.12.3
